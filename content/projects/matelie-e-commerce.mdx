# Matelie E-commerce

Challenge:

To develop an e-commerce using decoupled wordpress + nextjs for performance. Customer is selling artwork internationally.

## Used Tech

- TypeScript
- NextJS
- React
- Tailwind
- Pagar.me API (Payment Gateway)
- Newsletter API
- Wordpress (Auth + Blog API)
- Woocommerce (E-commerce API)
- HeadlessUI
- Heroicons
- Redux
- Algolia (full text search)
- DaisyUI
- SwiperJS (for carousels)
- Next-i18next (SSR Translations)
- Nextjs-progressbar
